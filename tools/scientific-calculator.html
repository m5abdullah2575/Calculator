Function Reference

Trigonometric

  • sin, cos, tan: Basic trig functions
  • sin⁻¹, cos⁻¹, tan⁻¹: Inverse functions
  • Note: Calculations in radians

Logarithmic

  • log: Base-10 logarithm
  • ln: Natural logarithm (base e)
  • e: Euler's number (≈2.718)

Other Functions

  • √: Square root
  • ∛: Cube root
  • x^y: Power function
  • n!: Factorial
  • π: Pi (≈3.14159)

How to Use the Scientific Calculator

  1. 1 Click number buttons to enter values
  2. 2 Use operation buttons (+, −, ×, ÷) for basic math
  3. 3 Click scientific function buttons for advanced operations
  4. 4 Press = to calculate the result
  5. 5 Use AC (All Clear) to start over or CE (Clear Entry) to clear current number
